## プロジェクト概要
webサイト構築, Github等の勉強かねて, npx4x4が個人的に欲しいと考え作ったツールとそれをまとめたサイトとなっています.

順に学習したほうが理解しやすいと考え, 高専の演習で使用したことのあるHTMLから始まり, 徐々に新たな技術を取り入れています.

使用技術の変遷は次の項で確認することができます.

## 各種ページ・ツールの紹介など
記載の順番はそのまま作成した時系列となっています.

### 1. ホーム(HTML, CSS)
タイトルと各種ツールへのリンクボタン, SNSアカウントへのリンクなどを設置しています.

### 2. 時刻指定タイマー(JavaScript)
終了時刻を決め, 到達までの残り時間を計るタイマーです.

視覚で把握しやすいバーもついており, 「○○時までに終わらせるタスク」などに利用すると便利です.

### 3. 文字数カウンター(JavaScript)
入力されている文字列をカウントするシンプルな文字数カウンターです.

### 4. 実寸シミュレータ(TypeScript)
表示領域の画素数と画面サイズの情報をもとに,　実寸サイズの画面を表示することができます.  
>[!CAUTION]
>ご使用のスクリーンより大きい画面サイズをシミュレーションすると見切れます.

ノーパソやスマホを買い替える際に画面のサイズ感確認するのに便利だと思います.

### 5. 席替えスロットル(TypeScript, React)
一席ずつスロットル形式で楽しく席替えを行えます.
大部分をVSCord拡張機能のGithubCopilotとGeminiを利用し作成しています.
