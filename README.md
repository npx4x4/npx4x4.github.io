## プロジェクト概要
フロントエンド, Github等の勉強かねて, npx4x4が個人的に欲しいと考え作ったツールとそれをまとめたサイトとなっています.
順に学習したほうが理解しやすいと考え, 高専の演習で使用したことのあるHTMLから始まり, 徐々に新たな技術を取り入れています.

使用技術の変遷は次の項で確認することができます.

## 各種ページ・ツールの紹介など
記載の順番はそのまま作成した時系列となっています.
()内は使用技術を表示です.
#### ホーム
タイトルと各種ツールへのリンクボタンが設置してあります.
SNSアカウントへのリンクなども置いてあります.
(HTML, CSS)
#### 時刻指定タイマー
終了時刻を決め, 到達までの残り時間を計るタイマーです.
視覚で把握しやすいバーもついており, 「○○時までに終わらせるタスク」などに利用すると便利です.
(HTML, CSS, JavaScript)
#### 文字数カウンター
入力されている文字列をカウントするシンプルな文字数カウンターです.
(HTML, CSS, JavaScript)
#### 実寸シミュレータ
表示領域の画素数と画面サイズの情報をもとに,　実寸サイズの画面を表示することができます.※ご使用の画面より小さくないと見切れます※
ノーパソやスマホを買い替える際に画面のサイズ感確認するのに便利だと思います.
(HTML, CSS, TypeScript, Node.js)
#### 席替えスロットル
スロットル形式で
楽しく席替えを行えます.
(CSS, TypeScript, Nodejs, React)
