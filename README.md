## プロジェクト概要
本プロジェクトは, webサイト構築やGithubの勉強などかねて, npx4x4が個人的に欲しいと考え作ったツールとそれをまとめたサイトとなっています.

順に学習したほうが理解しやすいと考え, 高専の演習で使用したことのあるHTMLから始まり, 徐々に新たな技術を取り入れています.

使用技術の変遷は次の項で確認することができます.

## 各種ページ・ツールの紹介など
記載の順番はそのまま作成した時系列となっています.

### 1. ホーム(HTML, CSS)
タイトルと各種ツールへのリンクボタン, SNSアカウントへのリンクなどを設置しています.

HTMLとCSSのみの簡素なものですが, デバイスによる表示ズレの調整に苦戦しました.

### 2. 時刻指定タイマー(JavaScript)
終了時刻を決め, 到達までの残り時間を計るタイマーです.\
「○○時までに終わらせるタスク」などに利用できるタイマーが欲しかったため作成しました.\
視覚で把握しやすい残り時間バーもついており便利かと思います.

基本は以前にProcessingにて構築したアプリの再現となっており, 相違は時刻入力の方法程度です.\
Processing版に関しては以下のQiita記事で確認できます.\
[Processingで時刻指定のタイマーをつくる](https://qiita.com/npx4x4/items/373c8c6709668d216916)

### 3. 文字数カウンター(JavaScript)
入力されている文字列をカウントするだけのシンプルな文字数カウンターです.\
便利な文字数カウントアプリはすでに多数存在しているため, 作成する必要性はないですが, jsの練習になるかと考え作成しました.

改行の除外や, 半角全角でカウント数を区別するなど, 定番の機能追加を考えています.

### 4. 実寸シミュレータ(TypeScript)
任意のサイズの画面(を模した図形)を実寸でシミュレートできます.\
ノートパソコンやスマホを買い替える際に, 画面のサイズ感を簡単に確認出来たらいいなと思い作成しました.
>[!CAUTION]
>ご使用のスクリーンより大きい画面サイズをシミュレーションすると見切れます.

画像埋め込みや円形への対応もできそうなので, 今後挑戦してみたいです.

### 5. 席替えスロットル(TypeScript, React)
スロット演出を1席ずつ確定させ, 楽しく席替えを行うことができます.\
既存の席替えサイトは演出がシンプルなため, より席替えをエンタメとして盛り上がれるツールが欲しいと思い作成しました.

本ルール作成にあたっては, UIが複雑になってきたため, aiに要件を伝えてtsxを書かせてみたところ, なんと処理部分まで実装したコードが出力されてきました.\
せっかくですので, AIを活用した開発に挑戦してみようと, 大部分をVSCord拡張機能のGithubCopilotとGeminiを利用し実装しました.
